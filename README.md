# AI 简历优化助手

这是一个基于 Python、Streamlit 和大模型 API 的 AI 项目。

用户输入岗位描述和简历内容后，系统会自动分析岗位匹配度，找出简历问题，给出修改建议，提取缺失关键词，并生成优化后的简历内容。

## 项目功能

- 岗位匹配分数评估
- 简历优点分析
- 简历问题诊断
- 修改建议生成
- 缺失关键词提取
- 优化后的简历生成
- 支持不同岗位类型
- 支持不同优化风格
- 支持下载优化结果

## 项目技术栈

- Python
- Streamlit
- OpenAI 兼容接口
- python-dotenv
